Upload SVG files in WordPress.

== Description ==

Using this simple Upload SVG image plugin you can allow SVG file format to your wordpress website. This is very simple and secure SVG plugin.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `asf-allow-svg-files.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
